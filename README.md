# ButtonPanel
Arduino code for button array for bistro

**Button arrangement**
```
LED
0 1
2 3
4 5
```

**Format**

On startup: ```start```

On button down: ```<buttonNum>D```

On button up: ```<buttonNum>U```

**Expected LED behaviour**

On button press led will light up until button is released

**Info**

Sample rate: 20ms
